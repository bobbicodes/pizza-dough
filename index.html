Pizza Dough Calculator

Input the number of cups of flour as indicated in the code below to compute your recipe: 

(defn tsp-salt [c-flour]
  (* 1/2 c-flour))
(defn tsp-sugar [c-flour]
  (* 1.45 c-flour))
(defn tsp-yeast [c-flour]
  (* 0.68 c-flour))
(defn pizza [c-flour]
  {:cups-of-flour c-flour
   :teaspoons-salt (tsp-salt c-flour)
   :teaspoons-sugar (tsp-sugar c-flour)
   :teaspoons-yeast (tsp-yeast c-flour)})

; change the following value to the number of cups of flour:

(pizza 1)
